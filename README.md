## ✨ Features

- **Dynamic Preview:** See exactly what your QR frame looks like in real-time.
- **Customization Options:**
  - 🎨 **Frame Styles & Themes:** Minimal, Bold, Gradient, Elegant, and Playful.
  - 🖼️ **Logo Integration:** Upload your own logo (with automatic aspect-ratio preservation and error-correction handling).
  - 🔠 **Typography & Colors:** Pick the exact fonts and colors that match your brand.
  - 📐 **Advanced Metrics:** Adjust QR size, corner radii, padding, text alignment, and more.
- **Multi-language Support:** Automatically detects your browser language (supports EN, PT, ES, FR, DE).
- **SEO Optimized:** Built with JSON-LD Schemas, Open Graph tags, and optimized for AI crawlers (ChatGPT, Claude, etc).
- **Dark/Light Mode:** Seamlessly switch between beautiful, modern UI themes.

## 🚀 Quick Start

To run this project locally, you will need **Node.js** installed.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Lickas/qrbuilder.git
   cd qrbuilder
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 🛠️ Technology Stack

- **Framework:** React + Vite
- **Language:** TypeScript
- **Styling:** Custom CSS (Modern UI, Glassmorphism, CSS Variables)
- **Icons:** Lucide React
- **QR Generation:** qrcode.react
- **Image Generation:** html-to-image
- **i18n:** react-i18next
